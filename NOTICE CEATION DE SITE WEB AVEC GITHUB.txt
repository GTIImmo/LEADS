📌 Guide Complet : Héberger ton Mini-Site HTML sur GitHub Pages
Tu veux héberger ton site web avec ton code HTML sur GitHub Pages. Je vais t’expliquer clairement et étape par étape ! 🚀

📌 1️⃣ Étape 1 : Créer un Dépôt GitHub (LEADS)
1️⃣ Va sur GitHub et connecte-toi
2️⃣ Clique sur "New repository"
3️⃣ Nom du repo : LEADS (ou un autre nom si tu préfères)
4️⃣ Coche "Public" (sinon GitHub Pages ne pourra pas afficher ton site)
5️⃣ Clique sur "Create Repository"

✅ Ton dépôt GitHub est prêt !

📌 2️⃣ Étape 2 : Préparer ton Code HTML sur ton Ordinateur
💡 Ton code HTML doit être dans un dossier propre.

1️⃣ Sur ton PC, crée un dossier pour ton site :
📁 C:\Users\frede\Desktop\LEADS

2️⃣ Mets ton fichier HTML principal dans ce dossier :

📄 index.html (obligatoire, c'est la page d'accueil)
📄 style.css (si tu as du CSS à part)
📄 script.js (si tu as un fichier JS à part)
📁 /images (si tu as des images)
💡 Assure-toi que ton fichier principal s'appelle index.html !

Modifier

2️⃣ Envoie tes fichiers sur GitHub

sh
Copier
Modifier

✅ Tes fichiers sont maintenant sur GitHub !

📌 6️⃣ Étape 6 : Activer GitHub Pages
💡 On va maintenant afficher ton site sur Internet !

1️⃣ Va sur ton repo GitHub : https://github.com/GTIImmo/LEADS
2️⃣ Clique sur "Settings" (⚙️ en haut à droite)
3️⃣ Dans le menu de gauche, clique sur "Pages"
4️⃣ Sous "Source", sélectionne la branche main et / (root)
5️⃣ Clique sur "Save"

📢 Après quelques minutes, GitHub va générer ton site !
